Une photo placeholder (SVG) est utilisée par défaut : yohan-placeholder.svg.

Pour mettre ta vraie photo :

1. Dépose ton fichier ici (yohan.jpg, yohan.png ou yohan.webp).
   Recommandé : carré, 400x400 px minimum, fond neutre.
   Le composant la rogne en cercle (object-fit: cover).

2. Mets à jour la balise <img class="about-photo"> dans
   src/routes/+page.svelte avec le bon chemin.
